# Cosmic Fallback Page 🌌

A next-generation, visually stunning 404/Fallback service page built with React, Vite, Tailwind CSS (v4), and Framer Motion. This page is specifically designed to elegantly inform users when a repository, app link, or demo page is temporarily unavailable. 

## ✨ Features

- **Advanced 3D Glassmorphism**: Hover-tracking 3D tilt effects using pure React state and Framer Motion logic.
- **Particle Engine**: Custom physics-driven particle backgrounds scaling seamlessly across displays.
- **Dynamic Glow Effects**: Ambient pulsing orbs and glowing lucide-react iconography.
- **Responsive Layout**: Carefully orchestrated mobile, tablet, and desktop breakpoints.
- **Hyper-Optimized**: Delivered lightning fast with Vite.

## 🚀 Quick Start

Ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Clone the repository** (or download the source):
   ```bash
   git clone <your-repo-link>
   cd Fallback
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   > Head over to `http://localhost:5173` to see it instantly.

## 🛠️ Tech Stack

- **React 18** (UI Library)
- **Vite** (Build Tool)
- **Tailwind CSS v4** (Utility-first CSS Framework)
- **Framer Motion** (Production-ready Animation Library)
- **Lucide React** (Beautifully crafted Open Source icons)

## 📦 Deployment (Netlify / Vercel / Pages)

This project has been pre-configured with a `public/_redirects` file designed for Netlify/Cloudflare, ensuring client-side React routing works immediately out of the box without 404 errors on refresh.

### Deployment settings:
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output/Publish Directory**: `dist`
- **Node Version**: Check that your host uses Node 18+ for best compatibility.
